# PPvIS_Lab 1
Simple simulation of animal world
## Options
  - add_object - add object on map by coords   
    - -o or --object - type of object(Plant, Ground, Herbivore or Predator) 
    - -x or --xcoord 
    - -y or --ycoord    
  - generate - generate new world
  - make_step
  - make_several_steps 
    - -n or --number - num of steps
  - show
