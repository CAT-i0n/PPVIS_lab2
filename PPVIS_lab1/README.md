# PPvIS_Lab 1
Simple simulation of animal world
## Options
  - add_object - add object on map by coords   
    - -o --object - type of object(Plant, Ground, Herbivore or Predator) 
    - -x --xcoord 
    - -y --ycoord    
  - generate - generate new world
  - make_step
  - make_several_steps 
    - -n -- number - num of steps
  - show
